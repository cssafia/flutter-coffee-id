# ☕ Coffee ID - Flutter Learning Project

This is a small Flutter application built while following the **Flutter tutorial series by Net Ninja**.
The goal of this project was to practice Flutter fundamentals such as layouts, widgets, state management, and reusable components.

## 📱 Features

* Display a coffee preference card
* Adjust coffee **strength**
* Adjust **sugar amount**
* Use **reusable widgets**
* Basic Flutter UI layout using:

  * `Scaffold`
  * `Column` / `Row`
  * `Container`
  * `Expanded`
* Image assets and styling

## 🧠 What I Learned

While building this project I practiced:

* Flutter project structure
* Stateless vs Stateful widgets
* Using `setState()` for UI updates
* Creating reusable components
* Working with assets (`pubspec.yaml`)
* Basic UI layout with Flutter widgets

## 📂 Project Structure (important folders)

```
lib/        -> Flutter source code
assets/     -> Images used in the UI
android/    -> Android platform files
ios/        -> iOS platform files
```

Generated folders such as `build/`, `.dart_tool/`, and `.idea/` are excluded from version control.
